## 核心改进

### 1. 多层次日志系统

**新增日志级别**：
- `[INFO]` - 绿色：一般信息
- `[WARN]` - 黄色：警告（不影响运行）
- `[ERROR]` - 红色：错误（可能导致失败）
- `[DEBUG]` - 青色：调试信息（--verbose 模式）
- `[STEP]` - 青色：步骤标记（步骤 n/6）
- `[  →]` - 紫色：子步骤标记
- `[✓]` - 绿色：成功标记
- `[PROGRESS]` - 蓝色：进度更新

### 2. 实时监控系统

**监控内容**：
- ✅ ROS2 节点状态（每 10 秒）
- ✅ ROS2 话题状态（每 10 秒）
- ✅ 建图进度追踪（每 10 秒）
- ✅ 系统资源监控（CPU/内存/GPU）
- ✅ 文件系统状态检查

**监控节点**：
- `automap_system` - 主建图节点
- `laserMapping` - Fast-LIVO 前端节点
- `rviz` - 可视化节点
- `rosbag_player` - Bag 播放节点

**监控话题**：
- `/livox/lidar` - 激光雷达点云
- `/livox/imu` - IMU 数据
- `/optimized_pose` - 优化后位姿
- `/submap_map` - 子地图
- `/global_map` - 全局地图

### 3. 自动诊断功能

**诊断项**：
1. 日志文件检查
2. 最新日志分析（错误、警告、完成状态）
3. 节点状态检查
4. 话题状态检查
5. 输出文件检查
6. 诊断建议

## 使用指南

### 快速开始

```bash
# Docker 模式（推荐）
./run_full_mapping_docker.sh -b @data/automap_input/nya_02.bag

# 本地模式
./run_full_mapping_enhanced.sh -b data/.../nya_02.bag

# 快速启动菜单
./quick_start_enhanced.sh
```

### 查看日志

```bash
# 查看日志摘要
./view_logs.sh

# 查看错误信息
./view_logs.sh -e

# 查看节点监控
./view_logs.sh -n

# 查看话题监控
./view_logs.sh -t

# 查看进度追踪
./view_logs.sh -p

# 运行诊断
./view_logs.sh -d

# 实时跟踪日志
./view_logs.sh -f
```
